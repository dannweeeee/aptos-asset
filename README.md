# Aptos Assets

This repository contains the template for minting tokens on Aptos.
- Aptos Fungible Asset Standard 
- Aptos Digital Asset Standard (NFT)

## Getting Started

First, install the dependencies:

```bash
brew install aptos
```

Then, run the following commands:

```bash
aptos init
```

```bash
aptos move compile
```

```bash
aptos move publish
```
